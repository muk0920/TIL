## 2-3 목록을 만드는 태그 

![image-20200521083037877](images/image-20200521083037877.png)

- `<ul>` : unordered list (순서x)
  - type="a" : 순서 목록을 알파벳으로 설정
- `<ol>` : ordered list (순서o)
  - start ="3"  : 숫자가 3부터 시작
